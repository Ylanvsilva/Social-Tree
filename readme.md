# Responsividade

## CSS Units

### Unidades de medidas do CSS

#### Layout Fixo

`px` - Pixels

#### Layout Fluido

`%` - Porcentagem

`auto` - Automática

`vh` - Viewport Height

`vw` - Viewport Width

#### Textos Fixos

`1px` = 0.75pt

`16px` = 12pt

#### Texto Fluidos

`em` - Multiplicado pelo pai

`rem` - Multiplicado pelo root

#

`html` `{`

    /* a cada 1rem será considerado 10px */
    
    font-size: 62.5%
`}`
